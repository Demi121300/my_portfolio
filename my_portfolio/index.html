IoT-Based Crop Watering System with Rain Detection and Remote Sensing Technology using Raspbery PI

For my capstone project, I developed an innovative IoT-based crop watering system designed to optimize agricultural irrigation. Using a Raspberry Pi 3B+, the system integrates rain detection and remote sensing technologies to monitor soil moisture levels and weather conditions. This approach ensures efficient water usage by automatically adjusting irrigation schedules based on real-time data, enhancing crop health and conserving resources.
